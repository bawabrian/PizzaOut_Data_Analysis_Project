Project Overview
Pizza Out is a fictional pizza company created for this data analysis project. The purpose of the project is to analyze sales data and present key business insights using Microsoft Excel only, focusing on clarity, structure, and practical business questions.

Objective
The goal of this project is to:
- Analyze sales performance for a pizza business
- Identify trends by day and hour
- Understand customer ordering behavior
- Evaluate product performance by size and category
- Practice building a structured, insight-driven dashboard in Excel

Tools Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers & Timeline
  - Calculated fields
  - Basic formatting and layout design

Dashboard Summary

Key Metrics
- Total Revenue: $817,860.05
- Total Orders Placed: 21,350
- Total Pizzas Sold: 49,574
- Average Order Value: $38.31
- Average Pizzas per Order: 2.3

Sales Trends
Daily Trends:
Orders increase steadily through the week, with Fridays and Saturdays being the busiest days.

Hourly Trends:
Most orders are placed in the evening hours, indicating strong dinner-time demand.

Sales Breakdown

By Pizza Size
Large pizzas account for the highest share of sales, followed by Medium and Regular sizes.

By Pizza Category
The Classic category is the most popular, followed by Chicken, Supreme, and Veggie.

Product Performance
Best Sellers:
- Classic Deluxe Pizza
- Barbecue Chicken Pizza
- Hawaiian Pizza

Worst Sellers:
- Brie Carre Pizza
- Mediterranean Pizza
- Calabrese Pizza

Interactivity
A timeline slicer allows filtering sales data by month to analyze trends across periods.

Data Notes
All data used is simulated. Pizza Out is fictional and used for demonstration purposes only.


Purpose of the Project
This project was created to practice Excel-based data analysis and dashboard design.
